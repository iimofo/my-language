# My Language Extension

This is a Visual Studio Code extension for the My Language programming language.

## Features

- Syntax highlighting for My Language files (.mylang extension)
- Basic language support for My Language
- Support for printing strings and numbers

## Usage

1. Install the extension from the [Visual Studio Code Marketplace](https://marketplace.visualstudio.com/items?itemName=your-username.my-language).
2. Open a file with a .mylang extension to activate the extension.
3. Start coding in My Language with syntax highlighting support.

## Requirements

- Visual Studio Code
- Node.js
